\noindent
{\Large \textbf{Aufgaben-Blatt}: Berechnung von erreichbaren Punkten}
\vspace{0.5cm}

\noindent
\textbf{Hinweis}: Versuchen Sie bei der Lösung der nachfolgenden Aufgaben
möglichst mit Mengen-Konstruktionen  und nicht mit Kontroll-Strukturen wie
 \texttt{for} oder \texttt{while} zu arbeiten.
\vspace{0.3cm}

\noindent
\textbf{Aufgabe 1}:  Gegeben sei eine binäre Relation $R$ auf einer Menge $M$.
Falls $\langle a, b \rangle \in R$ ist, sagen wir, dass es eine direkte Verbindung von dem
Punkt $a$ zu dem Punkt $b$ gibt.  Implementieren Sie eine Prozedur \texttt{oneStep},
so dass der Aufruf \\[0.1cm]
\hspace*{1.3cm} $\mathtt{oneStep}(x,R)$ \\[0.1cm]
für einen Punkt $x$ und eine binäre Relation $R$ alle die Punkte berechnet,
die von $x$ in einem Schritt erreicht werden können.

Implementieren Sie nun eine Prozedur \texttt{oneStepSet},
so dass der Aufruf \\[0.1cm]
\hspace*{1.3cm} $\mathtt{oneStepSet}(M,R)$ \\[0.1cm]
für eine Menge von  Punkten $M$ und eine binäre Relation $R$ alle die Punkte berechnet,
die von einem Punkt $x\in M$  in einem Schritt erreicht werden können.

\vspace{0.3cm}

\noindent
\textbf{Aufgabe 2}:  Gegeben sei wieder eine binäre Relation $R$ auf einer Menge $M$.
 Implementieren Sie unter \underline{Benutzun}g\underline{ von Auf}g\underline{abe 1} eine Prozedur \texttt{reachable},
so dass der Aufruf \\[0.1cm]
\hspace*{1.3cm} $\mathtt{reachable}(x,R)$ \\[0.1cm]
für einen Punkt $x$ und eine binäre Relation $R$ alle die Punkte berechnet,
die von $x$ nach beliebig vielen Schritten erreicht werden können. 

\noindent
\textbf{Hinweis}: Verwenden Sie für die Lösung einen Fixpunkt-Algorithmus, der
die folgende Gleichung löst: \\[0.1cm]
\hspace*{1.3cm}  $M = M + \mathtt{oneStepSet}(M,R)$.
\vspace{0.3cm}

\noindent
\textbf{Aufgabe 3}:  Gegeben ist nun  eine  \emph{Abstandsfunktion} $D$.  
Eine \emph{Abstandsfunktion} ist, wie in der Vorlesung, eine Menge von Elementen der Form
$\bigl\langle \langle a, b \rangle, l \bigr\rangle$, wobei $x$ und $y$ Punkte sind und $l$ den
Abstand zwischen diesen Punkten bezeichnet.  Implementieren Sie in Analogie zu Aufgabe
1 eine Funktion \\[0.1cm]
\hspace*{1.3cm} $\mathtt{oneStepSet}(M, R)$, \\[0.1cm]
die als Ergebnis eine Menge $E$ von Paaren der Form $\langle y, l\rangle$ berechnet.
Das Paar $\langle y, l\rangle$ soll genau dann in $E$ liegen, wenn es eine direkte
Verbindung von einem Punkt $x\in M$ zu dem Punkt $y$ gibt, so dass diese Verbindung die Länge $l$ hat.
\vspace{0.3cm}

\noindent
\textbf{Aufgabe 4}: Gegeben ist wieder eine  \emph{Abstandsfunktion} $D$.  
 Implementieren Sie unter Benutzung von Aufgabe 3 eine Prozedur \texttt{reachable},
so dass der Aufruf \\[0.1cm]
\hspace*{1.3cm} $\mathtt{reachable}(x,D)$ \\[0.1cm]
für einen Punkt $x$ und eine Abstandsfunktion $D$ eine Menge $E$ von Paaren der Form $\langle y, l\rangle$ berechnet.
Das Paar $\langle y, l\rangle$ soll genau dann in $E$ liegen, wenn es eine 
Verbindung von $x$ nach $y$ mit beliebig vielen Zwischenschritten gibt,  die insgesamt
die Länge $l$ hat und wenn es außerdem keine kürzere Verbindung gibt.
\vspace{0.3cm}

\noindent
Die folgende Aufgabe ist als Hausaufgabe für alle diejenigen gedacht,
die etwas sportlichen Ehrgeiz besitzen.
\vspace{0.1cm}

\noindent
\textbf{Aufgabe 5:} 
Ändern Sie das \underline{in Auf}g\underline{abe 4 erstellte Pro}g\underline{ramm} so ab, dass neben der Entfernung
auch noch ein kürzester Weg berechnet wird.  Schicken Sie das erstellte Programm dann an meine
Email-Adresse: \texttt{stroetmann\symbol{64}ba-stuttgart.de}.  Die beste Einsendung 
wird mit einer Flasche Sekt prämiert.
\vspace{0.1cm}

\noindent
\textbf{Hinweis}: Natürlich können Sie sich bei der Lösung an unserem Vorgehen bei der
Berechnung des transitiven Abschlusses orientieren und ähnlich wie dort eine Pfad-Relation
definieren.  Sie sollen aber \textbf{nicht} den transitiven Abschluss der Relation
berechnen, denn das ist zu aufwendig.
