\noindent
{\Large Wem gehört das Zebra?}
\vspace{0.5cm}

\noindent
Schreiben Sie ein \textsc{SetlX}-Programm, das das folgende Rätsel löst.
\begin{enumerate}
\item Es gibt 5 Häuser. Jedes der Häuser hat eine andere Farbe.
\item In jedem Haus wohnt ein Bewohner einer anderen Nationalität.
\item Jeder Hausbewohner bevorzugt ein anderes Getränk, raucht eine andere
      Marke Zigaretten und hält genau ein Haustier.
\item \underline{Keine} der Personen trinkt das Gleiche, raucht das Gleiche 
      oder hält die gleiche Art Haustier. 
\end{enumerate}
Außerdem wissen wir folgendes:
\begin{enumerate}
\item Der Brite wohnt im roten Haus.
\item Der Schwede hält einen Hund.
\item Der Amerikaner trinkt Whisky. 
\item Das grüne Haus steht links vom weißen Haus.
\item Der Besitzer vom grünen Haus trinkt Kaffee.
\item Die Person, die Camel raucht, hält einen Papagei.
\item Der Mann, der im mittleren Haus wohnt, trinkt Milch.
\item Der Besitzer vom gelben Haus raucht Dunhill.
\item Der Norweger wohnt im ersten Haus.
\item Der Marlboro-Raucher wohnt neben dem, der eine Katze hält.
\item Der Mann, der ein Schwein hält, wohnt neben dem, der Dunhill raucht.
\item Der Winfieldraucher trinkt gerne Bier.
\item Der Norweger wohnt neben dem blauen Haus.
\item Der Deutsche raucht Rothmanns.
\item Der Marlboro-Raucher hat einen Nachbarn, der Wasser trinkt.
\end{enumerate}
Die Frage lautet nun: Wem gehört das Zebra?
